    if n <= 1:

    return fibonacci(n-1) + fibonacci(n-2)

## 应用功能介绍

### 1. 文档管理
- 创建、编辑、删除文档
- 自动保存功能
- 文档版本管理

### 2. 搜索功能
- 全文搜索
- 标签过滤
- 智能匹配

### 3. 数据存储
- 本地 IndexedDB 存储
- 数据导入导出
- 离线可用

### 4. 用户界面
- 响应式设计
- 暗色/亮色主题
- 移动端适配

## 使用技巧

### 快捷键
- `Ctrl + S`: 保存文档
- `Ctrl + N`: 新建文档
- `Ctrl + F`: 搜索

### 标签使用
建议使用有意义的标签来组织文档：
- `技术` - 技术相关文档
- `笔记` - 学习笔记
- `项目` - 项目文档
- `想法` - 灵感记录

### 文档组织
- 使用清晰的标题结构
- 添加目录和索引
- 合理使用标签分类
- 定期整理和更新
