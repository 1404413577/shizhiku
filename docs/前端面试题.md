# 前端面试题

### 1.原型链是什么
- 核心功能是继承属性和方法的共享作用
- 每个对象都有一个内部属性  _proto_ 指向构造函数的prototype 属性，当一个新的对象去访问属性和方法的时候，优先从自身对象查找，如果没有就通过_proto_ 方向上查找，原型链的终端是null 就是 
  ```javascript
     object.prototype._proto_ === null 
  ```


### 2.new 操作符 
- new 核心的功能是创建构造函数的实例对象的操作符，与原型链构成联系，并完成实例化的初始化。
- 讲到new操作符，就要先了解一下this ,可以理解为一种绑定规则，但是并不会因为对象的创建就绑定在对象上，取决于对象调用的位置，但是用new操作符创建的对象的话，this就会强制绑定到这个对象的自身
- new还有一些特殊的行为，当构造函数返回的是对象或者函数的时候，，那么new操作符会直接返回这个返回值，如果构造函数返回的是普通类型，这个返回值会被忽略

### 3.从浏览器输入url到页面显示步骤发生了什么
- 关键字 dns域名解析 找到真实ip 向服务器发起请求 服务器返回数据 浏览器接收数据（html css js 静态文件）浏览器加载资源进行语法解析，渲染页面
