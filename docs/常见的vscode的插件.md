# 常见的vscode的插件

VS Code有许多实用的插件，以下是一些常见的插件：
- **语言支持类**：
    - **Chinese (Simplified) Language Pack for Visual Studio Code**：将VS Code的界面转换为中文，方便中文用户使用。
    - **Language Support for Java(TM) by Red Hat**：为Java开发提供支持，包括语法高亮、智能提示、调试等功能。
    - **Vue (Official)**：官方推出的Vue语言支持插件，为Vue单文件组件提供语法高亮、智能提示、代码补全、错误校验等功能。
- **代码格式化类**：
    - **Prettier - Code formatter**：一款超级受欢迎的代码格式化插件，可以统一代码风格，解决单双引号、括号换行等问题，还可设置为保存时自动格式化。
    - **Beautify**：可以美化JavaScript、JSON、CSS、Sass和HTML代码。
    - **ESLint**：将ESLint集成到VS Code中，用于检查和强制代码规范，确保代码质量。
- **代码补全与提示类**：
    - **Tabnine**：基于AI的智能代码补全插件，可学习编程习惯，智能预测接下来要写的代码，提高编码速度。
    - **Path Intellisense**：可自动填充文件名和路径，方便在导入模块和引用文件时使用。
    - **npm Intellisense**：用于在import语句中自动填充npm模块，提供包提示功能。
- **Git管理类**：
    - **GitLens**：增强了VS Code中内置的Git功能，如查看commits搜索、历史记录、代码作者身份等。
    - **Git Graph**：可以查看存储库的Git Graph，并从该图中轻松执行创建、删除、合并分支，提交代码等操作。
- **调试类**：
    - **Debugger for Chrome**：允许从VS Code调试在Google Chrome中运行的JavaScript代码，方便前端开发调试。
    - **Python Debugger**：支持使用debugpy进行Python调试，可设置断点、单步执行代码、检查变量等。
- **其他实用类**：
    - **Settings Sync**：使用GitHub Gist同步多台计算机上的VS Code设置，包括代码段、主题、文件图标等。
    - **filesize**：在状态栏中显示当前文件大小，点击后还可以看到详细的创建、修改时间。
    - **Live Server**：可以实时监听文件变化，保存后浏览器自动刷新，适合前端静态页面和小型项目开发调试。
    - **Todo Tree**：可以快速搜索工作区中的TODO和FIXME等注释标记，并在资源管理器窗格的树视图中显示它们。
